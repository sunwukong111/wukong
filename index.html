Core Sections

按企业官网的典型需求预留三个首屏下方模块：价值主张、解决方案分区与信任背书， 后续可继续向下延展案例、资源与表单。

I

品牌价值主张区域

以不超过三条的优势要点，强调产品或服务的核心价值。建议控制在 2–3 行文字内，避免与 Hero 文案重复，适合作为销售话术的默认版本。

适配：咨询 / 科技 / 金融 对齐业务负责人
II

解决方案 / 产品分区

将复杂产品折叠为 3–4 个清晰入口，配合简短描述与次级 CTA。 推荐按行业 / 功能 / 角色维度拆分，方便用户快速完成自我定位。

推荐：3–4 张卡片 映射产品矩阵
III

信任背书与社会证明

用于放置客户 Logo、评级、媒体报道或核心数据，构成“被市场验证”的证据。 建议保持极简布局，让 Logo 与数字自己说话。

组合：Logo + 数据 + 引言 对接品牌 & PR
